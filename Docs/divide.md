# divide

Number → Number → Number

## Function Parameters

(a)
The first value.

(b)
The second value.

Returns (Number) The result of `a / b`.

## Function Technical Explanation

Divides two numbers. Equivalent to a / b.

## Examples
```javascript
R.divide(71, 100); //=> 0.71

var half = R.divide(R.__, 2);
half(42); //=> 21

var reciprocal = R.divide(1);
reciprocal(4);   //=> 0.25
```
